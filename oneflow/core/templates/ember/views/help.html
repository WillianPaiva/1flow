{% filter markdown %}{% blocktrans %} ## Google Reader import This import takes places in the background. You can safely disconnect from **1flow** or close your brower window and come back later. Obviously, importing your data will take longer if many users import their own data at once. We have finites computing resources. This import status is updated every 5 seconds. The real number of imported articles will be less than the processed ones, due to some duplicates present on Google Reader. About the approximative ETA… Well… Given the nature of the data we fetch from Google, it's **very** approximative. Don't rely on it, we don't do. The import will probably end before the ETA. {% endblocktrans %}{% endfilter %}
